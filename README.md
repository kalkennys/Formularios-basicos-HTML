## 🎯 Sobre o Projeto

Este projeto foi criado para **iniciantes e intermediários** em HTML que querem:
- ✅ Aprender as principais tags de formulários
- ✅ Entender atributos essenciais para cada campo
- ✅ Conhecer boas práticas de desenvolvimento
- ✅ Ver exemplos práticos e testar interativamente

**Perfeito para:**
- 👨‍💻 Quem está começando a aprender HTML
- 📚 Estudantes de bootcamp (como DIO)
- 🎓 Material de referência rápida
- 🔗 Projeto open source para contribuir

## 📋 Tópicos Abordados

| Tag | Descrição | Importância |
|-----|-----------|------------|
| `<form>` | Container do formulário | ⭐⭐⭐ Crítica |
| `<input>` | Campos versáteis (10+ tipos) | ⭐⭐⭐ Crítica |
| `<label>` | Associa texto aos campos | ⭐⭐ Alta |
| `<select>` | Menus suspensos | ⭐⭐ Alta |
| `<textarea>` | Texto com múltiplas linhas | ⭐⭐ Alta |
| `<button>` | Botões interativos | ⭐⭐ Alta |
| `<fieldset>` | Agrupa campos | ⭐ Média |
| `<datalist>` | Autocomplete | ⭐ Média |
| `<optgroup>` | Agrupa opções | ⭐ Média |

### Atributos Essenciais
- `required` - Campo obrigatório
- `placeholder` - Texto de exemplo
- `pattern` - Validação por regex
- `name` - Identificador do campo
- `id` - Para CSS e JavaScript
