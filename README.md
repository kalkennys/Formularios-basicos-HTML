# 📚 Guia Completo de Formulários HTML

Um guia educativo e interativo sobre formulários HTML com as melhores práticas e tags mais usadas por desenvolvedores profissionais.

## 🎯 Sobre o Projeto

Este site foi criado para ajudar você a aprender:
- As principais tags de formulários HTML
- Atributos essenciais para cada campo
- Boas práticas de desenvolvimento
- Um exemplo funcional e prático de formulário

## 🚀 Como Usar

1. Abra o arquivo `index.html` em seu navegador
2. Explore cada seção para aprender sobre as tags
3. Teste o formulário de exemplo no final da página
4. Verifique o console do navegador para ver os dados capturados

## 📋 Tópicos Abordados

- **Tag `<form>`** - Container fundamental
- **Tag `<input>`** - Campo versátil com 10+ tipos
- **Tag `<label>`** - Melhora acessibilidade
- **Tag `<select>`** - Menus suspensos
- **Tag `<textarea>`** - Texto com múltiplas linhas
- **Tag `<button>`** - Botões interativos
- **Tags secundárias** - `<fieldset>`, `<datalist>`, `<optgroup>`
- **Atributos universais** - required, placeholder, pattern, etc.
- **Boas práticas** - Dicas de um desenvolvedor profissional
